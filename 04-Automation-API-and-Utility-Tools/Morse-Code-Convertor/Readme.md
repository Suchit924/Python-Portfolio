# 📡 Morse Code Converter

A Python tool to convert English text to Morse code and vice versa.

## ✨ Features
- Convert English → Morse
- Convert Morse → English
- Clean CLI interface

## 🛠️ Tech Used
- Python
- Dictionary mapping for Morse symbols

## ▶️ How to Use
1. Run the script:
   ```bash
   python main.py
   ```
